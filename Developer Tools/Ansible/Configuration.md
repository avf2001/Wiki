Лучше всего хранить все настройки в отдельной папке, а не полагаться на глобальные настройки.

```shell
ansible-config list
ansible-config dump
ansible-config dump --only-changed
ansible-config view
ansible-config init
```

# Inventory
```
/etc/ansible/hosts
```
# Configuration file
```
/etc/ansible/ansible.cfg
```
