## Editor
* Format Document (`Shift` + `Alt` + `F`) - Format the entire active file.
* `Alt` + Set cursor - Multiple cursors
* Select text + `F2` - Symbol rename
* `Alt` + &#8593; - Move line up
* `Alt` + &#8595; - Move line down
