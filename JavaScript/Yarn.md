# Обновление Yarn
```cmd
> npm install --global yarn
```
# Сборка проекта
```cmd
> yarn build
```
# Библиотеки
## Добавление библиотеки
```cmd
> yarn add @emotion/react
```
## Обновление библиотеки
```shell
> yarn upgrade @elastic/eui@48.1.0
> yarn upgrade @elastic/eui@48.1.0 # обновление библиотеки до определенной версии
```
