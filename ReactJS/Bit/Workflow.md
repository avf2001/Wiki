# 1. Создание рабочего пространства (workspace)
## 1.1. Вариант 1
Создание рабочего пространства
```cmd
> bit new react <my-workspace-name>
```
Справка по команде
```cmd
> bit new --help
```
## 1.2. Вариант 2 (предпочтительный)
```cmd
> mkdir <my-workspace-name>
> cd <my-workspace-name>
> bit init
```
